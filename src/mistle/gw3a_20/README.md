# NanoMig for the Gowin GW3A-LV20

This is a preliminary port of the NanoMig core to the [Gowin Arora III GW3A FPGA family](https://www.gowinsemi.com/en/product/detail/84/). It's meant to be used on our [upcoming custom GW3A-LV20 development board](https://github.com/MiSTle-Dev/Boards/tree/main/mistle_gw3a_20).
